# 插件

插件的开发和使用自小程序基础库版本 [1.9.6](../compatibility.html "基础库 1.9.6 开始支持，低版本需做兼容处理。") 开始支持。

插件是对一组 js 接口或[自定义组件](../custom-component/)的封装，用于提供给第三方小程序调用。插件必须嵌入在其他小程序中才能被用户使用。

插件开发者可以像开发小程序一样编写一个插件并上传代码，在插件发布之后，其他小程序方可调用。小程序平台会托管插件代码，其他小程序调用时，上传的插件代码会随小程序一起下载运行。

相对于普通 js 文件或自定义组件，插件拥有更强的独立性，拥有独立的 API 接口、域名列表等，但同时会受到一些限制，如[一些 API 无法调用或功能受限](api-limit.html)。

对于插件开发者，请阅读[开发插件](development.html)章节；对于插件使用者，请阅读[使用插件](using.html)章节。

</section>
