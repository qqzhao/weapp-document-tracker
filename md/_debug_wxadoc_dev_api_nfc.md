### wx.onHCEMessage(CALLBACK)

> 基础库 1.7.0 开始支持，低版本需做[兼容处理](../framework/compatibility.html)

监听 NFC 设备的消息回调，并在回调中处理。返回参数中 `messageType` 表示消息类型，目前有如下值：

*   1：消息为HCE Apdu Command类型，小程序需对此指令进行处理，并调用 `sendHCEMessage` 接口返回处理指令；
*   2：消息为设备离场事件
