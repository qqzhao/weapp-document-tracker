# 插件调用 API 的限制

插件可以调用的 API 与小程序不同，主要有两个区别：

*   插件的请求域名列表与小程序相互独立；
*   一些 API 不允许插件调用。

目前，允许插件调用的 API 及其对应版本要求如下。
